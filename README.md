# E-Commerce Management

A minimal full-stack e-commerce web app with React/Vite frontend and Express backend.

## Live Demo

- Frontend + API on Vercel: https://ecommerce-web-theta-three.vercel.app

## Repository

- GitHub: https://github.com/Periyannan35/E-Commerce-management

## Features

- Product browsing
- User authentication
- Cart and checkout flow
- Order history
- Admin product management
- JSON-backed server data store (no MongoDB required)

## Local setup

```bash
cd "E_comerce Web/server"
npm install
node index.js
```

Then open the client separately from its folder:

```bash
cd "E_comerce Web/client"
npm install
npm run dev
```

## Vercel deployment

Configured with `vercel.json` to deploy the static frontend and server API together.
